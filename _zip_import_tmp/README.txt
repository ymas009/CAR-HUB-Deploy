Pune to Maharashtra Travel UI Pack

This pack is made only for Maharashtra visiting places from Pune.

Files:
- src/components/MaharashtraDestinations.jsx
- src/components/MaharashtraDestinations.css
- src/assets/maharashtra-travel/*

Install:
npm install lucide-react

Use in App.jsx or Home.jsx:
import MaharashtraDestinations from "./components/MaharashtraDestinations";

export default function App() {
  return <MaharashtraDestinations />;
}

Places included:
- Pune to Lonavala
- Pune to Mahabaleshwar
- Pune to Alibaug
- Pune to Matheran
- Pune to Lavasa
- Pune to Nashik
- Pune to Shirdi
- Pune to Kolad

If you want realistic images, use IMAGE_PROMPTS_MAHARASHTRA.txt and replace the SVG files with your PNG/JPG images.
