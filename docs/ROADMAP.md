# CarHub Roadmap

## MVP

- Public package discovery.
- Customer registration/login foundation.
- Step-based package request form.
- Admin request review and approval workflow.
- Provider onboarding and assigned-work panel.
- Provider-visible shared payload model.
- Support ticket foundation.
- Feedback moderation foundation.
- Audit logging.
- PostgreSQL schema and Spring Boot API structure.
- Responsive animated React UI foundation.

## Growth Stage

- Real payment integration.
- Notification service with email/SMS/WhatsApp providers.
- Provider quality score automation.
- SLA dashboards and escalation automation.
- Refund/reschedule workflows.
- CDN-backed media management.
- Redis caching and async queue processing.
- More comprehensive E2E tests.

## Enterprise Stage

- MFA for admins and support.
- SSO support.
- Fine-grained permission administration.
- Compliance exports and audit retention policies.
- CRM, analytics, support desk, and map integrations.
- Multi-language and multi-currency support.
- Read replicas and stronger DR planning.

## Ultra-Scale Stage

- Domain service decomposition.
- Event streaming for workflow/audit/notifications.
- Data partitioning by region/time.
- Multi-region traffic and failover.
- Advanced fraud/anomaly detection.
- Customer mobile apps on the same API platform.
